Step 1 - Initial Site Content

One of the longest and most tedious parts of this project in my opinion. This part includes the selection of the technology, the design of the website as well as the website content.

Overall Technology:

Python – Django – HTML – Javascript/JQuery – SCSS/CSS – Bootstrap 3 – GitHub

Step 2 - Design & Implementation of Content Database

Having my own website is nice an all, but, in the end, it's really just static content created by front-end technology. I'll be transfering all the content to a database to make it more manageable and organised.

This step is really just a preparation for the next step (creating the CMS) due to the various kinds of web content.

Step 3 - Implement CMS for the Content

This is where the complexity gets real. An interface will be created in order to be able to manage all the content of this website. The features I currently have in mind include the following:

  • > Adding permissions
  • > Customizing attributes of a page
  • > Creating and deleting content and pages
